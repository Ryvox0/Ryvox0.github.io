PulseEvents Documentation

Installation • Commands • Permissions • System Overview

📦 Overview

PulseEvents is an automated Minecraft event system that handles event scheduling, random selection, announcements and BossBar countdowns.

⚙ Installation

1. Download plugin

2. Put into plugins/

3. Restart server

4. Configure /plugins/PulseEvents/

🧭 Commands

/pe help — Commands list
/pe start — Start event
/pe stop — Stop event
/pe status — System status
/pe reload — Reload config
/pe update — Check updates

🔐 Permissions

pulseevents.admin
Full access
pulseevents.use
Basic usage

⚡ Features

  • Auto event system
  • BossBar countdown
  • Random events
  • Multi-world support
  • Anti-spam system

⚠ Notes

  • Events must be pre-configured
  • No in-game event creation
  • Still under active development
