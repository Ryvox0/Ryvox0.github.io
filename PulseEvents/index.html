PulseEvents

Automated Minecraft event system with BossBar countdown & random events

Spigot • Paper • Java 17+

Overview

PulseEvents automatically manages Minecraft server events using a random event system, announcements, and BossBar countdowns — fully server-driven.

Features

⚡ Auto event system
🎯 BossBar countdown
🎲 Random event selection
🌍 Multi-world support
🔐 Permission system
🧠 Anti-spam cooldowns

Commands

/pe start — start event
/pe stop — stop event
/pe status — check system
/pe reload — reload config
/pe update — check updates

Planned Features

🔮 In-game event builder
📊 PlaceholderAPI support
💰 Vault integration
🎬 ActionBar / Titles
