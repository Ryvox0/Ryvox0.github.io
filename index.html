⚡ PulseEvents

Automated Minecraft Event System with BossBar Countdown

📦 About

PulseEvents is a lightweight Minecraft plugin that automatically manages server events, including announcements, countdowns, and random event execution. if you want to send idea click the button 💡 Send Idea

⚡ Features

  • Automatic event system
  • BossBar countdown
  • Random event selection
  • Multi-world support
  • Command system

📊 Roadmap
